# 指尖花园 - 美甲预约小程序

微信小程序美甲预约项目，可用微信开发者工具导入当前目录运行。

## 功能

- 查看未来 10 天可预约日期
- 按“基础项目 / 设计效果 / 装饰”多选美甲项目
- 自动显示已选项目摘要和预估价格
- 选择空闲预约时段，已占用时段自动置灰
- 客户填写姓名、微信号或手机号、备注后提交预约
- 客户可在“我的”页面按联系方式查询和取消预约
- 店主可登录后台查看预约、取消预约、设置休息时段
- 店主可设置单个休息时段或全天休息，占用后客户侧不可预约

## 云开发连接

当前版本通过 `cloudfunctions/quickstartFunctions` 接入微信云开发。

云数据库集合：

- `appointments`：预约和店主休息占用数据
- `owners`：店主账号数据

首次启动会调用云函数自动创建 `appointments` 和 `owners` 集合，并初始化默认店主账号。具体初始化账号配置在云函数中维护，不在文档中暴露。

如果项目只有一个默认云环境，`app.js` 中无需额外配置。若有多个云环境，请在 `app.js` 里设置：

```js
cloudEnvId: "你的云环境 ID"
```

## 部署步骤

1. 在微信开发者工具中导入当前目录。
2. 确认 `project.config.json` 中的 `appid` 是你的小程序 AppID。
3. 开通或选择已配置好的云开发环境。
4. 右键 `cloudfunctions/quickstartFunctions`，选择“上传并部署：云端安装依赖”。
5. 编译小程序，进入预约页测试提交预约。
6. 在云开发数据库中确认 `appointments` 和 `owners` 集合已创建。

## 店主登录

店主账号由云函数初始化到 `owners` 集合。小程序前端不保存明文密码，登录时通过云函数校验账号密码。

如需更换店主账号或密码，请修改云函数中的初始化配置后重新部署云函数；已生成到 `owners` 集合的数据可在云数据库中按需更新或删除后重新初始化。
