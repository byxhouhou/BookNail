# 指尖花园 - 美甲预约小程序

这是一个微信小程序美甲预约项目，可直接用微信开发者工具导入当前目录运行。

## 功能

- 查看未来 10 天可预约日期
- 按“基础项目 / 设计效果 / 装饰”多选美甲项目
- 自动显示已选项目摘要和预估价格
- 选择空闲预约时段
- 已预约时段自动置灰，避免重复预约
- 客户填写姓名、微信号或手机号、备注后提交预约
- 客户可在“我的”页面按联系方式查询和取消预约
- 店主可用“店主”入口登录，查看全部预约并取消预约
- 店主可设置休息时段，占用对应日期和时间，客户侧不可预约

## 云开发连接

当前版本已通过 `cloudfunctions/quickstartFunctions` 接入微信云开发，预约数据保存到云数据库 `appointments` 集合。

首次启动会调用云函数自动创建 `appointments` 集合，并写入当天两个演示占用时段。

如果你的项目只有一个默认云环境，`app.js` 中无需额外配置。若有多个云环境，请在 `app.js` 里设置：

```js
cloudEnvId: "你的云环境 ID"
```

## 部署步骤

1. 在微信开发者工具中导入当前目录。
2. 确认 `project.config.json` 中的 `appid` 是你的小程序 AppID。
3. 开通或选择已配置好的云开发环境。
4. 右键 `cloudfunctions/quickstartFunctions`，选择“上传并部署：云端安装依赖”。
5. 编译小程序，进入预约页测试提交预约。
6. 在云开发数据库中查看 `appointments` 集合是否新增记录。
